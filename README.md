# Trae AI Agent 自定义智能体描述合集

[Trae 官网](https://www.trae.ai/)

## 项目简介

本项目用于整理和沉淀可直接用于 Trae 的中文智能体描述（Prompt / Agent Description）。

当前仓库以文档为主，核心内容收录在 `Agent_Description.md` 中，涵盖开发、运维、学习辅导与专业翻译等多个场景，适合作为：

- 自定义智能体的基础提示词模板
- 不同岗位 AI 助手的人设与职责说明
- 多场景智能体配置素材库

## 项目结构

```text
AI_Agent/
├── Agent_Description.md   # 智能体详细描述与职责定义
└── README.md              # 项目说明文档
```

## 文档特点

- 按“身份 + 职责 + 规则”方式组织，便于直接复用
- 覆盖技术栈、工作边界、输出风格和行为约束
- 适合继续扩展更多垂直领域智能体

## 使用方式

你可以将 `Agent_Description.md` 中对应角色的内容直接复制到 Trae 或其他支持自定义系统提示词的平台中，作为智能体的基础描述。

建议使用流程：

1. 根据使用场景选择目标智能体
2. 按需裁剪或补充行业规则、输出格式、禁用项
3. 在实际对话中持续迭代 Prompt 内容
